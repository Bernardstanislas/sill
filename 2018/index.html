Présentation

Le Le Socle interministériel des logiciels libre contribue au cadre de convergence des logiciels à privilégier dans le développement des systèmes d'information de l'Etat, il présente une liste de logiciels libres à utiliser de préférence, de manière concertée et coordonnée.

Cette page non officielle propose une exploration dynamique de la liste.

Vous pouvez améliorer le contenu de cette page via son dépôt git et/ou proposant une question/suggestion.

Pour toute question, écrire à Bastien Guerry.

Retour au site principal.

Statut

  • R = recommandé
  • O = en observation
  • FV = en fin de vie

Colonne Windows

  • 2 = versions x86 et x64
  • x = non précisé

Colonne Linux

  • version distribution = version courante de la distribution Linux utilisée
  • i = inutile ; fonctionnalité existante nativement

Les lignes en italique correspondent à des extensions, des modules complémentaires ; le logiciel parent est alors indiqué dans la colonne "Logiciel parent"
