Présentation

Le Socle interministériel des logiciels libres est une liste de logiciels libres servant à la modernisation des systèmes d'information de l'État.

Cette page non officielle propose une exploration dynamique de la liste.

Vous pouvez améliorer le contenu de cette page via son dépôt git et/ou proposant une question/suggestion.

Pour toute question, écrire à Bastien Guerry.

Retour au site principal.
